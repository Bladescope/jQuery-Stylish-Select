jQuery-Stylish-Select by brettohland

stylishSelect turns that drab select box into a delicious looking list of elements based on your browser width.

Demo

Click on the select box and be filled with amazement as a beautifully styled and columned box appears

The columns will restack based on the width of the browser window, pretty nifty eh?

There are a few styles that need to be set to control how the box looks, for example:

        
          .selectionList {
            background: #81D8D0;
            width: 600px;
            position: absolute;
            padding: 30px;
            margin-top: 5px;
          }
          .selectionList ul {
            display: block;
            float: left;
            margin-right: 20px;
            margin-left: 0;
            padding: 0;
            list-style: none;
          }
          .selectionList p {
            margin: 0 0 0 5px;
            text-transform: uppercase;
          }
          .selectionList li { margin-bottom: 5px; }
          .selectionList li a {
            color: #000;
            text-decoration: none;
            padding: 3px 5px;
            width: 130px;
            display: block;
            cursor: pointer;
          }
          .selectionList li a:hover { text-decoration: underline; }
          .selectionList.small   { width: 200px; }
          .selectionList.medium  { width: 500px; }
          .selectionList.large   { width: 700px; }
